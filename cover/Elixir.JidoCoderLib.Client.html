3 Client API for session management and interaction.

5 This module provides a clean, high-level API for clients (TUI, web, API, etc.)

7
8 The Client module is stateless and delegates to:

13 ## Architecture
14
15 The Client API serves as the public interface for all client types:
16
17 ```
18 ┌─────────────────────────────────────────────────────────────┐
19 │ CLIENT LAYER │
20 │ TUI │ Web │ API │ Custom - all use Client API │
21 ├─────────────────────────────────────────────────────────────┤
22 │ JidoCoderLib.Client (this module) │
23 │ - Session lifecycle (create, terminate, list, get_info) │
24 │ - Message routing (send_message) │
25 │ - Event subscription (subscribe_to_session) │
26 ├─────────────────────────────────────────────────────────────┤
27 │ Internal Agents (not accessed directly) │
28 │ SessionManager │ ContextManager │ PubSub │
29 └─────────────────────────────────────────────────────────────┘
30 ```
